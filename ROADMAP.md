# Omni IDE — Product Roadmap 🗺️

Our goal is to build the world's most capable autonomous engineering environment.

## 🟢 Phase 1: Foundation (Current - v3.0.0)
- [x] **Omni-Agent Core**: Task planning and multi-file editing.
- [x] **Self-Healing Terminal**: Basic crash-fix loop integration.
- [x] **Local Inference**: Support for Ollama and local LLMs.
- [x] **Windows Native**: Stable build for Windows developers.

## 🟡 Phase 2: Security & Scale (Q3 2026)
- [ ] **Omni-Guard Alpha**: Isolated Docker-based sandboxing for agent execution.
- [ ] **macOS & Linux Support**: Cross-platform parity.
- [ ] **Multi-Agent Orchestration**: Specialized agents for Backend, Frontend, and DevOps.
- [ ] **Gemini 1.5 Pro Integration**: Ultra-long context for massive codebase refactoring.

## 🔴 Phase 3: Autonomous Maturity (2027)
- [ ] **Deterministic Verification**: Formal verification of AI-generated patches.
- [ ] **Autonomous CI/CD**: Agents that manage their own deployment pipelines.
- [ ] **Omni-Cloud (Optional)**: Highly secure, encrypted cloud environments for heavy compute.
- [ ] **Enterprise Governance**: SOC2 compliant agentic workflows.

---
*Roadmap is subject to change based on community feedback and AI research breakthroughs.*
