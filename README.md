# Omni IDE — Landing Page 🚀

The official landing page for **Omni IDE**, an AI Research & Engineering company based in Bengaluru, India.

Built for maximum performance and visual impact, this single-file landing page is designed for immediate deployment to **GitHub Pages**.

## ✨ Features

- **3D Atmospheric Background**: Built with Three.js (Particles & Wireframes).
- **Agentic Hero Section**: Interactive elements with GSAP animations.
- **Agentic Ecosystem**: Cards for Omni-Guard, Omni-Agent, and Self-Healing Terminal.
- **MSME Verified**: Registered under UDYAM-KR-03-0681404.
- **Privacy First**: Zero telemetry, zero cloud lock-in.
- **Custom Cursor**: Smoothly interpolated dual-element cursor system.
- **SEO Optimized**: Open Graph, Twitter Cards, and JSON-LD structured data.

## 🛠️ Tech Stack

- **Core**: Vanilla HTML5 / CSS3 / JavaScript (ES6+)
- **Graphics**: [Three.js](https://threejs.org/)
- **Animations**: [GSAP](https://greensock.com/gsap/) (ScrollTrigger)
- **Typography**: [Google Fonts](https://fonts.google.com/) (Syne, DM Sans, JetBrains Mono)
- **Deployment**: [GitHub Pages](https://pages.github.com/)

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/nihannihu/landing-page.git
   ```
2. **Open `index.html`**:
   Simply open the file in any modern browser. No build steps required.
